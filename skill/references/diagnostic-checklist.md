# 诊断检查清单

> 从 SKILL.md 拆出的完整 P1 检查项，诊断模式按需查阅。

## P0 必查项（每次诊断都输出）

| 检查项 | 规则来源 | 适用类型 |
|--------|---------|---------|
| 字数是否超载体约束 | 载体约束表（`carrier-limits.md`） | [全部] |
| 虚词框架是否正向 | 虚词选择引擎（`function-words-table.md`） | [全部] |
| 按钮是否即动作 | "按钮即动作"原则，非抽象确认词 | [UI] |
| 是否有空洞形容词 | "很安全""性能优异""已优化"等无信息量表达 | [全部] |
| 是否误用"我"/"我们" | UI/MKT 禁用"我"，"我们"仅限 EXP/合规/安全 | [UI] [MKT] |
| 标点使用是否恰当 | UI 不用句号；感叹号 UI 禁用（庆祝场景可用1个）、MKT 利益场景克制使用、EXP 禁用；EXP 必须用句号 | [全部] |
| 中英文标点混用 | 中文语境使用中文标点（`grammar-templates.md`§十六） | [全部] | LM 已知常见规则，重点检查中文引号""和全角括号（） |
| 包容性语言 | 禁用词清单 + 感官动词替换（`grammar-templates.md`§十） | [全部] | 重点检查：黑/白名单、杀进程、盲区、疯狂 |
